本人相对来说更喜欢简约的博客主题，简单并且功能齐全，这样看着比较舒心，观感也好。这次我就总结说一下我的博客是怎么做成的吧，由于本人还是新手，有很多东西都不了解，如果文章中有任何错误，请指出来，我会积极学习改正的。

首先感谢国外大佬开发的hexo主题Cactus，我看中它的原因就是界面设置非常简洁，几乎没有任何多余的界面，麻雀虽小五脏俱全，很适合我这样专注于写作的博主了。

跟着github上的教程很快我就本地部署完成了，然后我就上传的自己的github仓库上搭建网站。跟着网上的教程也是简单易懂噢。

在测试工程中唯一让我耗费时间的是评论系统，Cactus主题自带的评论插件只有两个，一个是Disqus，一个是Utterances，Disqus链接的github仓库上的Discussions系统，但是没有回复功能，Utterances功能很齐全但是在国内被墙了看不到。所以我费尽心思寻找各种评论插件，有些还需要服务器做后端我就没用。最后找到了类似于Disqus的升级版gitcus，然后又遇到了很多麻烦，前面说过这个hexo主题只预装了两个评论系统，等于说自己还要在此基础上添加新的功能。对于刚接触这个的萌新的人来说可太麻烦了，看着自己根目录下的密密麻麻的文件夹不禁让我陷入了沉思，我先是问了一下gpt，用命令行在根目录下输入安装代码：
npm install giscus
